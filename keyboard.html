Javascript Piano Keyboard from freeCodeCamp

Modified from Joe Liangs project

I copied this piano keyboard in Javascript based on the dynamically generated synthetic keyboard made by Keith William Horwood. Joe Liang extended the number of keys available to 4 octaves, set new key bindings, applied some stylistic changes, and posted this to www-freecodecamp-org. To keep things simple, he removed the option of switching to different instrument sounds and just kept piano. There was also no CSS file, as Joe had it as a script in the HTML file. I added one and plan on getting the other instrument functionality back at some point.

What makes this keyboard interesting is the fact that Javascript dynamically generates the appropriate sound when the user presses a key and the code does not load any external audio files.
